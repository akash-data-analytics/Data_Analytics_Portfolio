# Customer Churn Analysis

## Project Overview

Customer churn analysis helps businesses understand why customers stop using their services. 
In this project, I analyzed a telecom customer churn dataset to identify patterns and factors 
that influence customer attrition.

The goal of the analysis is to discover insights that can help companies reduce churn 
and improve customer retention.

## Dataset

Dataset used: IBM Telco Customer Churn Dataset.

The dataset contains information about telecom customers such as:

- Customer demographics
- Services subscribed
- Contract type
- Payment method
- Monthly charges
- Customer churn status

## Tools and Technologies

- Excel – Data cleaning and preprocessing
- SQL – Data analysis and querying
- Python (Pandas, Matplotlib, Seaborn) – Exploratory Data Analysis
- Power BI – Interactive dashboard visualization

## Project Workflow

1. Data Cleaning
   - Removed missing values
   - Checked data types
   - Prepared dataset for analysis

2. SQL Analysis
   - Queried churn distribution
   - Analyzed churn by contract type
   - Analyzed churn by payment method

3. Python Exploratory Data Analysis
   - Visualized churn patterns
   - Compared churn vs monthly charges
   - Analyzed service usage patterns

4. Power BI Dashboard
   - Built an interactive dashboard
   - Visualized churn rate
   - Displayed churn by contract and payment type

   ## Key Insights

- Customers with month-to-month contracts show higher churn rates.
- Customers with higher monthly charges are more likely to churn.
- Customers using fewer services tend to leave the service earlier.

## Dashboard

The Power BI dashboard provides an overview of customer churn patterns, 
including churn rate, contract type distribution, and monthly charge analysis.
